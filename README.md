A Time-Based Memory Matching Game

Chrono Card is a browser-based memory matching game that challenges players to test their memory and speed under time pressure. The game features structured level progression, increasing difficulty, and persistent progress tracking, making it both engaging and fair.


 Features:

Time-based gameplay
Memory card matching mechanics
Sequential level unlocking
Star-based performance rating
Sound effects for enhanced feedback
Progress saved using localStorage
Clean and responsive user interface


 Technologies Used:
HTML – Page structure and layout
CSS – Styling, animations, and responsive design
JavaScript – Game logic and interactivity
localStorage – Persistent level progression
Web Audio API – Sound effects

 How to Play:
Start the game from the main menu
Select an unlocked level
Flip two cards at a time to find matching pairs
Match all pairs before the timer runs out
Complete levels to unlock new challenges


Level Progression: 
Levels unlock only after successful completion
Players cannot skip levels
Progress is saved automatically in the browser
